## 零、全局用法（生成前必读）

### 0.1 风格锚句（每条提示词前复制粘贴一次）

**中文风格锚句：**

> 2D 卡通 Q 版，西部沙龙与扑克赌徒主题，粗黑色闭合描边，平涂色块，简单赛璐璐一层阴影，暖土色与牛仔蓝主色，高剪影可读，游戏立绘素材，纯白背景，无文字无水印，非照片非写实

**英文锚句（部分模型对英文更稳时可整段贴）：**

> chibi western saloon poker gambler game art, thick black outlines, flat colors, simple cel shading, warm dusty palette, clean readable silhouette, full body or prop sheet, solid white background, no text no watermark, not photorealistic

### 0.2 通用负面（中文，可接在提示词末尾）

> 不要照片质感，不要细线插画，不要文字水印签名，不要畸形手指，不要多余肢体，不要模糊低分辨率，不要整图强烈镜头畸变，不要可读英文单词与招牌字母

### 0.3 推荐工作流

| 步骤 | 说明 |
|------|------|
| 1 | 文生图得到 **白底** 或 **透明底** 单张概念 |
| 2 | 角色/道具用 **抠图 / 移除背景**；序列帧用 **同 seed 锁人物** 或 ControlNet 线稿控形 |
| 3 | 导入 Godot：`Sprite2D` / `AnimatedSprite2D` / `NinePatchRect` / `TileSet` 按工程路径归档到 `assets/` 或 `scenes/` 下约定目录 |
| 4 | 与主角线宽对比：若偏细，在引擎里 **描边后处理** 或统一 **外描边叠加层** |
| 5 | **超长横图**（如 6000px）：优先 **分块生成同风格** 再拼接，或对单张 **超分** 后手动修接缝 |

### 0.4 分辨率与比例（便于直接设画布）

| 品类 | 建议比例 / 单边像素 | 备注 |
|------|---------------------|------|
| 基础敌人单帧 | 方图 **512～768** 或竖图 **512×640** | 与主角同量级可 **768**；碰撞圆心对齐脚底或几何中心 |
| 序列帧条带 | 单帧 **256～512** 方格 × N 帧 | 与现有主角 **320 格** 对齐亦可 |
| **横版长卷地图「淘金公路」** | **6000×1300** 整图，或 **3 张 2000×1300** 横向拼接 | **前景 / 中景 / 后景** 见 §二.1；拼接时锁定 seed 与调色 |
| 其他大地图块 / 远景 | **1024×1024** 可平铺贴图，或 **1920×1080** 单张远景 | 逻辑地图可大于单张贴图，用 **tile 重复** |
| UI 条/框 | **高 64～128** 或 **宽 512～1024** 横条 | 适合 **九宫格** 切边；导出 **PNG + alpha** |
| 弹道精灵 | **64×64～128×128** | 小而清晰；拖尾可用单独条带图 |
| 爆炸单帧 / 短序列 | **256×256～512×512** | 圆形占画面约 **60～80%** 直径便于对齐逻辑半径 |
| **扑克牌面（平面）** | 竖向约 **2.5:3.5**，短边 **512～1024** | 见 **§五**；圆角矩形、透明或白底按工程约定 |

## 一、基础敌人（定案：史莱姆、负鼠 + 可选方案）

**用途**：替换 `Enemy.tscn` 程序圆点占位；需 **idle**（及后续 **move / hurt / death** 可同锚句扩展），**敌对**气质，与主角 **线宽 / 色温** 一致。

### 1.0 敌人共通设计要点

- **轮廓**：缩略图 64px 仍能辨物种；避免与主角橘猫 **同色块糊成一团**。  
- **配色**：史莱姆偏 **金绿半透明 + 暖色核**；负鼠偏 **灰褐毛 + 脏粉耳/鼻 + 深色眼罩带**（神似强盗面罩，呼应赌徒）。  
- **西部符号**：可含 **迷你扑克花色**、**筹码**、**金粒** 等点缀，**小面积** 即可，忌全身贴满花纹。  
- **导出**：立绘 **纯白底** 或 **透明底**；动画序列 **每帧同画布尺寸**。

---

### 1.1 史莱姆（基础史莱姆）

**设计要点**：常规史莱姆形象，整体圆润、半透明胶质，底部扁圆贴地，无特殊装饰，颜色可为绿色或蓝色，表情简单带有高光，无明显四肢，可略有小突起作为伪足，形象友好但带有敌对气质。

**单段完整中文提示词（可直接复制）：**

> 2D 卡通 Q 版，常规史莱姆小怪形象，圆润半透明胶质身体，颜色为绿色或蓝色，底部扁圆贴地，整体造型简洁无复杂装饰，双眼带白色高光，表情简单可带微笑或坏笑，无明显四肢，仅两侧略有小突起作为伪足，游戏立绘素材，纯白背景，无文字无水印，非照片非写实，适合横版卷轴敌人角色剪影清晰

**负面补充（接末尾）：**

> 不要复杂头饰或道具，不要奇异颜色，不要多眼多嘴，不要拟人化手脚

**英文备用：**

> chibi slime enemy, classic round shape, translucent green or blue gel body, no extra decorations, simple oval base, bright highlight eyes, side view idle, white background, game sprite

**动画扩展（move / 受击，锚句同上后接）：**

- **移动**：身体横向压扁和拉长交替，底部贴地滑行，两侧小突起模拟伪足晃动。  
- **受击**：整体颜色微闪白，身体稍变形或凹陷一帧，表现受击感。

### 1.2 负鼠（西部公路负鼠）

**设计要点**：四足站立 **略驼背** 像小偷；**深灰褐皮毛**，**浅粉长鼻** 与 **圆耳朵**；眼部 **黑色强盗式眼罩纹**（自然毛色形成，非人类面具）；**长尾卷曲** 可缠一枚 **小筹码** 或 **半张扑克**；姿态 **鬼祟** 不卖萌。

**单段完整中文提示词（可直接复制）：**

> 2D 卡通 Q 版，西部沙龙与扑克赌徒主题，粗黑色闭合描边，平涂色块，简单赛璐璐一层阴影，暖土色与牛仔蓝主色，高剪影可读，游戏立绘素材，纯白背景，无文字无水印，非照片非写实，一只敌对负鼠小怪四足站立侧视，深灰褐色皮毛腹部略浅米白，浅粉色长鼻尖与圆耳朵内侧浅粉，双眼周围自然形成深色强盗式眼罩毛色，小黑豆眼带狡黠眼神，长尾卷曲末端松松套一枚金色扑克筹码，穿一件故意做小的棕色皮马甲敞开不系扣露出肚皮，单帧待机姿势身体略前倾像准备偷牌，全身无复杂道具背景纯白，适合横版卷轴敌人

**负面补充（接末尾）：**

> 不要写实负鼠照片质感，不要血腥尖牙，不要巨型体型超过画面三分之一高度，不要飞行姿态

**英文备用：**

> chibi opossum bandit, masked eye fur, curled tail with poker chip, tiny leather vest, sneaky idle, thick outlines, western cartoon enemy, white bg, game sprite

## 二、地图设计

### 2.1 淘金公路（定稿横版长卷）

**规格（与美术定案对齐）**

| 项 | 数值 / 说明 |
|----|----------------|
| **画幅** | **宽 6000px × 高 1300px**（整卷）；若模型单张上限不足，拆为 **3×（2000×1300）** 左中右拼接，接缝放在 **中景建筑稀疏处** |
| **透视** | **横版侧视 / 略俯视**，地平线约在画面 **上 35%～40%** 高度，路面占 **下 45%～55%** |
| **分层语义** | **后景**：远山、峡谷轮廓、晚霞天空、远矿镇剪影；**中景**：双车道土公路、车辙、仙人掌、废弃木矿车、歪倒路牌、电线杆剪影；**前景**：近侧路缘碎石、尘土、半截铁轨、近景枯草与 **小淘金盆与散金粒**（不挡角色可走区域，宜画面 **最下 10%～15%** 条带） |
| **叙事关键词** | 淘金热、66 号公路气质土路、废弃矿井、扑克霓虹 **不出现文字** 可用 **模糊色块灯** |

**分轨生成建议（推荐，便于 Godot Parallax）**

1. **后景层**：单独一张 **6000×900** 或 **6000×1300** 仅天空+远山+远镇（路面留白透明或浅色平涂底便于抠）。  
2. **中景层**：**6000×1300**，路面+中距物体，**Alpha 通道** 或清晰地平线与前景分离。  
3. **前景层**：**6000×1300**，仅最前碎石、草、淘金盆等，**下半透明或硬边** 与角色排序约定。

## 五、扑克牌面（平面卡牌模板）

**用途**：单张牌 **正面** 美术资源；程序可换花色点数，本段以 **黑桃 A** 为 **构图与风格定调模板**，其它牌面复用同一套边框与底材质感。

**设计要点**

- **版式**：竖向 **圆角矩形** 扑克牌比例（约 **2.5:3.5**）；**经典边角**：左上/右下 **A** 与 **黑桃** 对称小标识，中心留白或极简主花（本模板为 **黑桃 A**）。  
- **风格**：**2D 平面**、**赛璐璐平涂**、**粗黑轮廓**；**复古西部赌场** 老扑克气质，**低饱和暖色**、**做旧纸张**；**高对比线稿**，避免花哨渐变。  
- **边框**：外层 **深蓝色斜纹肌理** + **边缘磨损泛黄、毛边**；内侧 **细黑圆角轮廓线** 与底色分离。  
- **底色**：中间大面积 **米黄 / 旧牛皮纸**，**轻微污渍与纸张纹理**，**干净留白** 供 UI 叠字（若工程需要再叠，生成图本身可无印刷数字）。  
- **导出**：单张 **PNG**；建议单边 **512～1024** 短边，便于九宫格或整图缩放。

### 单段完整中文提示词（可直接复制）· 黑桃 A 模板
