## 三、卡牌逻辑测试用例（v0.2）

### 顺序与分组

1. 起点为 `Q, K, A` → 合法顺子，长度 3。  
2. 起点为 `A, 2, 3` → 非法顺子 → 退化为单张组。  
3. `7,7,7,7` → 四条，长度 4。  
4. `5,5,6,7,8` 从索引 0：以「最长合法组」为准；期望须在实现注释中写死（顺子 `5,6,7,8` 或对子优先等）。  
5. 打出一组后，下一判定从**紧邻下一张**开始，不得跨组复用牌。  

### 出牌循环

6. 整副牌本轮打完（最后一组打出）→ 进入装配/重组计时（与 `CardRuntime` 一致；**不**以弹道命中为界）。  
7. 同一次遍历内步与步之间遵守基础出牌间隔。  

### 花色 — 本次攻击

8. 一组中某花色张数 ≥2 → 触发该花色本次效果。  
9. 满足花色阈值且**整组同花色** → 本次花色效果强化。  
10. 单张组不触发「多张才触发」的本地花色阈值类效果（按具体字段实现）。  

### 花色 — 全局

11. 全牌组内心张数达 3/6/9/12 时分别解锁对应档位。  
12. 各花色阈值**独立**：增加梅花档不得改变红心档。
