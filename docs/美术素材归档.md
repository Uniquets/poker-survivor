# 美术素材归档

与仓库内实际路径对齐，记录 **已有** 与 **缺失 / 待升级** 素材。美术需求与命名以 `详细设计.md`（含第二部内容规格）为准。

**维护约定**：新增或替换素材后，同步更新本表「路径 / 用途 / 状态」三列；**画面风格**以本章第一节为验收口径，避免与主角画风漂移。

**AI 生成专用**：基础敌人、地图、玩家信息 UI、手牌栏、**扑克牌面（§五）**、点数 2/8 弹道与爆炸等 **文生图中文提示词与工作流** 见 **`AI美术素材生成提示词.md`**（与第一节锚句一致）。

## 一、画面风格指引（全局统一）

以下词汇与约束**适用于**角色、敌人、场景、UI、特效与图标；外包或自制前先对照本节。

### 1.1 主题关键词（叙事与符号）

| 中文 | 说明 |
|------|------|
| **西部** | 荒漠、小镇、沙龙、酒馆、通缉令、马刺、尘土、黄昏天光 |
| **扑克** | 四花色、点数、牌背纹样、筹码、牌桌绿呢/木纹、出千与「读牌」隐喻 |
| **牛仔** | 宽檐帽、丹宁、皮带扣、领巾、马刺靴、实用主义剪裁 |
| **赌徒** | 骰子、筹码堆、Ace 标记、霓虹灯牌、高风险高对比的视觉暗示 |

### 1.2 画风关键词（对齐主角序列帧）

| 中文 | 英文参考 | 说明 |
|------|-----------|------|
| **2D 卡通** | cartoon 2D | 非写实比例，偏 Q 版、头大身小可接受 |
| **粗描边** | thick outline / clean lineart | **黑色或深褐闭合线**，线宽与主角同一量级，全项目统一 |
| **平涂为主** | flat color | 大色块填充，**少用**复杂体积渐变与过多渐变层 |
| **简单赛璐璐** | simple cel shading | **1～2 层**明确阴影即可，阴影边缘硬、形状几何化 |
| **高可读剪影** | readable silhouette | 缩略图仍能辨职业与阵营；敌我轮廓语言区分明显 |
| **暖土色调** | warm earth tones | 棕、赭、牛仔蓝、铁锈红为**主色**；点缀赌场红、扑克黑 |
| **纸品质感** | paper / cardstock | 牌面、UI 卡片区可带轻微纸纤维或压纹暗示，忌塑料高光 |
| **做旧可控** | subtle wear | 磨损、卷边、污渍**点到为止**，保持清爽可读 |

### 1.3 色彩与光照（全画面）

- **整体**：黄昏至油灯夜之间的**暖环境**；冷色仅作小面积点缀（如牛仔蓝、牌背深蓝）。  
- **对比**：关键交互（选中牌、危险区、弹道）用**提高明度或描边**拉开，避免靠脏灰降对比。  
- **饱和度**：中高饱和、**避免灰泥感**；暗部仍保持色相倾向，不偏纯黑死灰。

### 1.4 场景与特效

- **场景**：远景可略虚、块面化；**中近景 props** 线宽与角色一致，避免照片拼贴感。  
- **特效**：轮廓清晰、**几何化**（星芒、火花、纸牌轨迹）；忌写实烟雾粒子堆砌盖读性。

### 1.5 UI 与字体

- **形态**：木牌、蜡封、金属铆边、通缉令边框等**成套组件**；圆角与内边距全项目统一一档。  
- **字体**：标题偏 **Western / 海报木刻感**；正文偏 **易读无衬线或窄衬线**；数字可用 **赌场霓虹或筹码刻字** 变体，与正文字体**同一家族体系**。

### 1.6 禁忌（避免风格漂移）

- 写实 PBR、照片材质、**细线插画**、日系赛璐璐多层渐变主导  
- 高饱和霓虹铺满全屏、与西部叙事**无关**的科幻硬表面  
- 与主角**线宽 / 色温**明显冲突的素材混用（若必须第三方，需后期统一描边与调色）

### 1.7 外包/沟通用短语（可复制）

`Western pulp + poker saloon + chibi cowboy gambler cat, thick black outlines, flat colors, simple cel shade, warm dusty palette, readable game art, no photorealism`

## 三、缺失或待升级美术素材

按模块列出；**待升级** 指已有占位但与主题/量产规格不符，需替换或扩展。

### 1. 主角与战斗表现

| 状态 | 内容 |
|------|------|
| 缺失 | 受击 / 死亡 / 无敌表现（序列帧或骨骼）；施法或「出牌」前摇；冲刺/翻滚（若玩法需要） |
| 缺失 | 与弹道逻辑对应的 **飞牌 / 骰子 / 筹码** 等飞行体精灵或序列帧（含拖尾、命中爆点） |
| 待升级 | 主角 **分层部件**（帽、马甲、领巾、腰带、尾饰骰子）便于换装或染色 |
| 缺失 | **HUD 小头像**、选牌/结算用头像（与立绘同一比例体系） |

### 2. 敌人与 BOSS

| 状态 | 内容 |
|------|------|
| **已定案（AI 提示词见 `AI美术素材生成提示词.md` §一）** | **西部淘金史莱姆**、**公路负鼠** — 立绘/序列帧待生成入库 |
| 缺失 | 上述定案敌人的 **死亡 / 受击 VFX**；更多普通怪变体（郊狼、老千、人形 J/Q/K 等见 AI 文档 **§1.3 可选**） |
| 缺失 | 精英 / BOSS 造型与阶段变化（警长、赌场老板、机械老虎机等） |

### 3. 场景与环境

| 状态 | 内容 |
|------|------|
| **已入库（主场景默认底图）** | **`assets/sprites/scene/desert.png`**：原图 **8192×3520 px**，`RunScene/DesertMap` **`scale` 0.3**；逻辑边界 **`CombatTuning.MAP_*`** = 原尺寸 ×0.3（**2457.6×1056**） |
| **已定案（AI 提示词见 `AI美术素材生成提示词.md` §二.1）** | **「淘金公路」横版长卷**：**6000×1300 px**，**前景 / 中景 / 后景** 分层（可整卷或 3×2000×1300 拼接 + Godot 视差） |
| 缺失 | 与淘金公路混用的 **地面 tileset**、局部 **props** 独立切图（木桶、路牌等） |
| 缺失 | 其他主战斗 **大背景**（小镇主街 / 沙龙室内等）变体 |
| 可选 | 全局色调 LUT 或渐变遮罩（黄昏 / 油灯感） |

### 4. 扑克 UI 与牌面

| 状态 | 内容 |
|------|------|
| 待升级 | **全套牌面**（52+鬼或规则子集）统一画风，替换当前单张 SVG 占位 |

## 五、格式与工程建议（供导出约定）

| 项 | 建议 |
|----|------|
| 角色序列帧 | PNG + 透明；统一画布倍数；命名与 `SpriteFrames` 动画名（如 `idle` / `move`）一致 |
